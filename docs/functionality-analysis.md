# RSS Impact 项目功能分析

## 项目概述

RSS Impact 是一个支持 Hook 的 RSS 订阅工具，由前端（rss-impact-web）和后端（rss-impact-server）组成。该工具允许用户订阅 RSS 源，并在新文章发布时通过各种 Hook 机制触发自定义操作，如推送通知、Webhook 调用、文件下载等。

## 核心功能

### 1. RSS 订阅管理

**描述**：用户可以添加、管理和组织 RSS 订阅源。
**核心程度**：核心功能
**相关页面**：

-   订阅管理页面（Feed Management）
-   分类管理页面（Category Management）
    **相关接口**：
-   `/feed` - 订阅 CRUD 操作
-   `/category` - 分类 CRUD 操作

### 2. Hook 触发系统

**描述**：当 RSS 源有新文章时，自动触发预设的 Hook 操作，包括推送通知、Webhook、下载、AI 总结等。
**核心程度**：核心功能
**相关页面**：

-   Hook 管理页面（Hook Management）
-   Webhook 日志页面（Webhook Log）
    **相关接口**：
-   `/hook` - Hook CRUD 操作
-   `/webhook-log` - 日志查询

### 3. 文章管理与查看

**描述**：显示从订阅源获取的文章内容，支持查看和删除。
**核心程度**：核心功能
**相关页面**：

-   文章管理页面（Article Management）
    **相关接口**：
-   `/article` - 文章查询和删除

## 辅助功能

### 4. 用户认证与管理

**描述**：支持用户注册、登录、个人信息管理。多用户系统，支持管理员和普通用户角色。
**核心程度**：辅助功能
**相关页面**：

-   登录页面（Login）
-   注册页面（Register）
-   个人中心页面（Personal Center）
    **相关接口**：
-   `/auth` - 认证相关
-   `/user` - 用户管理

### 5. 数据导入导出

**描述**：支持 OPML 文件导入导出订阅列表，自定义查询生成 RSS 输出。
**核心程度**：辅助功能
**相关页面**：

-   自定义查询页面（Custom Query）
    **相关接口**：
-   `/custom-query` - 自定义查询管理

### 6. 代理与配置

**描述**：支持代理配置以访问受限 RSS 源。
**核心程度**：辅助功能
**相关页面**：

-   代理配置页面（Proxy Config）
    **相关接口**：
-   `/proxy-config` - 代理配置管理

### 7. 资源管理

**描述**：管理通过 Hook 下载的文件和资源。
**核心程度**：辅助功能
**相关页面**：

-   资源管理页面（Resource Management）
    **相关接口**：
-   `/resource` - 资源查询

### 8. 系统管理

**描述**：管理员功能，包括用户管理、系统统计、日统计等。
**核心程度**：辅助功能
**相关页面**：

-   系统管理页面（System）
-   日统计页面（Daily Count）
-   用户管理页面（User Management，仅管理员）
    **相关接口**：
-   `/system` - 系统信息
-   `/daily-count` - 统计数据
-   `/user` - 用户管理（管理员）

### 9. 其他页面

**描述**：首页、文档、关于等信息页面。
**核心程度**：辅助功能
**相关页面**：

-   首页（Home）
-   文档页面（Docs）
-   关于页面（About）
-   404 页面（Page404）

## 业务流程

1. **用户注册登录**：新用户注册账号，登录系统。
2. **订阅管理**：用户添加 RSS 订阅源，并分配到分类。
3. **Hook 配置**：为订阅源配置 Hook，定义触发条件和操作。
4. **文章获取**：系统定期获取 RSS 源更新，存储新文章。
5. **Hook 触发**：新文章触发相应 Hook，执行推送、下载等操作。
6. **数据查看**：用户查看文章、日志、资源等。

## 技术架构

-   **后端**：NestJS 框架，提供 RESTful API。
-   **前端**：Vue.js 框架，使用 Avue 组件库。
-   **数据库**：支持多种数据库（SQLite、MySQL 等）。
-   **部署**：支持 Docker 容器化部署。

## 总结

RSS Impact 的核心价值在于将 RSS 订阅与自动化 Hook 机制结合，实现 RSS 内容的智能处理和分发。核心功能围绕 RSS 订阅管理和 Hook 触发展开，其他功能为用户管理和系统配置提供支持。
